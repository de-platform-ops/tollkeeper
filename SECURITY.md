# Security Policy

## Scope

This policy covers the `write-audit-publish` library and the `airflow-wap` package in this repository.

## Reporting a vulnerability

Do not open a public GitHub issue for security vulnerabilities.

Report privately using one of these channels:

- GitHub's private security advisory feature: go to the Security tab on this repository, then "Report a vulnerability".
- Email: srchilukoori@gmail.com

Include enough detail to reproduce the issue: affected version, a minimal repro, and the impact you expect.

## Response time

You will get a response within 72 hours.

## Supported versions

This project is pre-1.0. Fixes land on `develop` and ship in the next release; there is no separate maintenance branch.
